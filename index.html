スプレッドシート 最強ショートカット 20選

脱・初心者！マウスを卒業して、作業時間を半分にするための厳選ガイド。
※ Mac版をメイン表記、( )内はWindows版の操作です。

01. 基本の必須操作

⌘ + C
Win: Ctrl + C

コピー

選択したセルの内容や数式をクリップボードに記憶します。

⌘ + V
Win: Ctrl + V

貼り付け

コピーした内容を、選択したセルへ流し込みます。

⌘ + Z
Win: Ctrl + Z

元に戻す

操作をミスしても大丈夫。直前の状態へ一歩戻ります。

02. 爆速移動と範囲選択

⌘ + 矢印
Win: Ctrl + 矢印

データ端へ移動

空白セルに当たるまで、データの端っこへ一瞬でジャンプします。

⌘ + Shift + 矢印
Win: Ctrl + Shift + 矢印

データ端まで選択

今のセルからデータの終わりまでを、一気に範囲選択します。

⌘ + A
Win: Ctrl + A

表全体を選択

カーソルがある場所に関連する表の範囲をすべて選択します。

⌘ + Home
Win: Ctrl + Home

先頭(A1)へ移動

シートのどこにいても、一瞬で左上のA1セルへ戻ります。

⌘ + F
Win: Ctrl + F

検索

膨大なデータの中から、特定のキーワードを素早く探します。

03. 入力・編集マジック

Enter
Win: F2 または Enter

セル内を編集

マウスでダブルクリックせずに、セルの文字を書き換え始めます。

⌘ + D
Win: Ctrl + D

下方向にコピー

上のセルの内容を、今のセル（または選択範囲）に複製します。

⌘ + R
Win: Ctrl + R

右方向にコピー

左のセルの内容を、今のセルへ一瞬で複製します。

⌘ + ;
Win: Ctrl + ;

今日の日付を入力

カレンダーを確認せずとも、一瞬で今日の日付を刻印します。

⌘ + K
Win: Ctrl + K

リンクの挿入

ウェブサイトのURLやシート内リンクをスマートに設定します。

04. 行と列の自由自在な操作

⌘ + Space
Win: Ctrl + Space

列全体を選択

今いる場所の「縦一列」をすべて選択状態にします。

Shift + Space
Win: Shift + Space

行全体を選択

今いる場所の「横一列」をすべて選択状態にします。

⌘ + Option + =
Win: Ctrl + Alt + =

行・列を挿入

選択している場所に、新しく空白の行や列を割り込ませます。

⌘ + Option + -
Win: Ctrl + Alt + -

行・列を削除

不要になった行や列を、跡形もなくスッキリ消去します。

05. シート管理とヘルプ

⌘ + Shift + PgDn/Up
Win: Ctrl + Shift + PgDn/Up

シートの切り替え

隣のタブへ移動します。マウスを使わず別シートへ移動できます。

⌘ + /
Win: Ctrl + /

ショートカット一覧

「何だっけ？」と思ったらこれ。全ての答えが画面に現れます。
