スプレッドシート 厳選ショートカット

初心者が最初に覚えるべき必須操作をまとめました。
※ Mac版をメイン表記、( )内はWindows版の操作です。

重要：ショートカットが効かない場合は、メニューの「ヘルプ」→「キーボード ショートカット」から
「対応するスプレッドシートのショートカットを有効にする」をONにしてください。

01. 基本の必須操作

⌘ + C
Win: Ctrl + C

コピー

選択したセルをコピーします。

⌘ + V
Win: Ctrl + V

貼り付け

コピーした内容を貼り付けます。

⌘ + X
Win: Ctrl + X

切り取り

セルの内容を切り取ります。

⌘ + Z
Win: Ctrl + Z

元に戻す

直前の操作をキャンセルします。

⌘ + Y
Win: Ctrl + Y

やり直し

「元に戻す」をさらに取り消します。

02. 爆速移動と範囲選択

⌘ + 矢印
Win: Ctrl + 矢印

データ端へ移動

データが入っている一番端のセルまでジャンプします。

⌘ + Shift + 矢印
Win: Ctrl + Shift + 矢印

範囲選択

現在のセルからデータ端までを一気に選択します。

⌘ + A
Win: Ctrl + A

全選択

表全体、またはシート全体を選択します。

Shift + Space
Win: Shift + Space

行を選択

現在の行を横一列すべて選択します。

03. 管理とヘルプ

⌘ + /
Win: Ctrl + /

ショートカット一覧

すべてのショートカットを確認できます。設定の有効化（ON）もここから行います。
